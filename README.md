# Stichting Patsboem Static Website

Dit is een static one-page website voor Stichting Patsboem.

## Bestanden
- `index.html` — hoofdwebsite
- `styles.css` — styling
- `script.js` — mobiel menu, FAQ en demoformulier

## Publiceren via GitHub Pages
1. Maak een nieuwe GitHub repository aan.
2. Upload deze drie bestanden.
3. Ga naar **Settings → Pages**.
4. Kies branch `main` en folder `/root`.
5. Sla op. GitHub geeft daarna een publieke website-link.

## Publiceren via Netlify
1. Ga naar Netlify.
2. Kies **Add new site → Deploy manually**.
3. Sleep de map of ZIP naar Netlify.
4. De site staat direct online.

## Formulier werkend maken
Het contactformulier is nu een demo. Koppel het later aan:
- Netlify Forms
- Formspree
- Google Forms
- eigen backend/API

## Aanpassen
Zoek in `index.html` naar:
- `info@patsboem.org`
- `Paramaribo, Suriname`
- teksten zoals missie, programma’s en doelstellingen

Pas deze aan naar jullie officiële gegevens zodra de stichting is geformaliseerd.
